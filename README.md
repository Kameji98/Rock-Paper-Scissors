# Rock Paper Scissors (Python)

A menu-driven CLI implementation of Rock Paper Scissors with clean input validation and score tracking.

## Features
- User vs. computer gameplay
- Validated input (`r`, `p`, `s`)
- Emoji output for a friendly CLI experience
- Score tracking (wins / losses / ties)
- Play multiple rounds and exit anytime

## Tech Stack
- Python 3 (standard library only)

## Run Locally
```bash
python src/main.py
```

## What I practiced
- Control flow and validation loops
- Small, readable functions
- Basic game state (score) management
